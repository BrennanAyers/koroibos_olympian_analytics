## Database Models
Our main storage of information is done through the Athletes table. An Athlete resource currently has the following required attributes:
- Name `String`
- Sex `String`
- Age `Integer`
- Height `Integer`
- Weight `Integer`
- Team `String`
- Games `String`
- Sport `String`
- Event `String`

And the following non-required attribute:
- Medal `String`
